主要用于展示大量结构化数据。

  • 当有大量结构化的数据需要展现时；
  • 当需要对数据进行排序、搜索、分页、自定义操作等复杂行为时。

代码示例

基础用法

表格的最简单用法。默认自带斑马纹，可通过表格参数striped控制。默认自带分页，可通过表格参数pagination控制。


<table id="table1"></table>
                
$('#table1').table({
    pagination: false,//取消分页
    columns: [{
        field: 'id',
        title: 'Item ID'
    }, {
        field: 'name',
        title: 'Item Name'
    }, {
        field: 'price',
        title: 'Item Price'
    }],
    data: [{
        id: 1,
        name: 'Item 1',
        price: '¥ 1'
    }, {
        id: 2,
        name: 'Item 2',
        price: '¥ 2'
    }, {
        id: 3,
        name: 'Item 3',
        price: '¥ 3'
    }]
});
无边框表格

默认情况下，表格是有边框的，可以通过定义表格参数classes添加 table-no-bordered类 来删除表格的边框样式。 


<table id="table2"></table>
                
$('#table2').table({
    classes: 'table table-hover table-no-bordered',
    pagination: false,//取消分页
    columns: [{
        field: 'id',
        title: 'Item ID'
    }, {
        field: 'name',
        title: 'Item Name'
    }, {
        field: 'price',
        title: 'Item Price'
    }],
    data: [{
        id: 1,
        name: 'Item 1',
        price: '¥ 1'
    }, {
        id: 2,
        name: 'Item 2',
        price: '¥ 2'
    }, {
        id: 3,
        name: 'Item 3',
        price: '¥ 3'
    }]
});
可排序表格

表格的分页分为服务端和客户端两种形式，排序亦然。 


<table id="table3"></table>
                
$('#table3').table({
    pagination: false,
    sidePagination:'client',
    columns: [{
        field: 'id',
        title: 'Item ID',
        sortable: true
    }, {
        field: 'name',
        title: 'Item Name',
        sortable: true
    }, {
        field: 'price',
        title: 'Item Price',
        sortable: true
    }],
    data:[
        {
            "id": 0,
            "name": "Item 0",
            "price": "$0"
        },
        {
            "id": 1,
            "name": "Item 2",
            "price": "$1"
        },
        {
            "id": 2,
            "name": "Item 1",
            "price": "$2"
        }
    ]
});
